### Módulos relacionados:

### Descripción del trabajo:
El proyecto consiste en desarrollar una plataforma web para la gestión integral del mantenimiento de un vehículo. Permitirá a los usuarios registrar sus coches, añadir mantenimientos, controlar gastos y consultar estadísticas.
